React组件生命周期

对组件从第一次实例化到最后销毁组件的整个生命周期的控制。
分为三阶段：

  1. 是组件第一次渲染阶段，在这里完成了组件的初始化和加载；(实例化)
  2. 是组件在运行和交互阶段，这个阶段组件可以处理用户交互，或者接收事件更新界面，以及因父组件的重新渲染而造成的变化；(存在期)
  3. 是组件卸载消亡的阶段，这里做一些组件的清理工作。(销毁期)
