JavaScript 基础之数据类型

2017年3月13日

JavaScript数据类型

JavaScript 主要有三大类数据类型：基础数据类型、引用数据类型、特殊数据类型。

基础数据类型

  • 字符串
  • 数值
  • 布尔

字符串

字符串数据类型用来表示JavaScript中的文本。在JavaScript 中，用双引号（””）和单引号(‘’)均可表示字符串。一般使用双引号(“”)来表示字符串被认为是最佳的，但具体使用视团队编码规范而定。

一个字符串值是排在一起的一串零或零以上的Unicode字符（字母、数字和标点符号）。

Unicode为每个字符都提供了唯一的数值，不管是什么平台、什么程序或什么语言。开发unicode是为了给处理世界上存在的所有字符提供统一的编码。

数值

JavaScript内部将所有的数值表示为浮点值，因此，在JavaScript中整数和浮点值没有差别。

Boolean数据类型

布尔（逻辑）只能有两个值：true 或 false。

特殊数据类型

  • null
  • undefined

null

可以通过一个变量赋值为null值来清除变量的内容。

需要注意在JavaScript中typeof运算符在判断null时会判断为Object,而不是null。

null用来表示尚未存在的对象，常用来表示函数企图返回一个不存在的对象。

undefined

以下情况将会返回undefined值：

  • 对象属性不存在
  • 声明了变量但从未赋值

引用数据类型（数组、对象）

JavaScript中引用数据类型是保存在堆内存中的对象，JavaScript不允许直接访问堆内存空间中的位置和操作堆内存空间，只能通过操作对象在栈内存中的引用地址。所以引用类型的数据，在栈内存中保存的实际上是对象在堆内存中的引用地址。通过这个引用地址可以快速查找到保存在堆内存中的对象。

在ECMAScript中，变量可以存在两种类型的值，即原始值和引用值。

原始值存储在栈（stack）中的简单数据段，也就是说，它们的值直接存储在变量访问的位置。引用值存储在堆（heap）中的对象，也就是说，存储在变量处的值是一个指针（point），指向存储对象的内存处。

简而言之，堆内存存放引用值(具体对象)，栈内存存放固定类型值。

数据类型转换和基本包装类型

JavaScript 为基本数据类型（string、number、boolean）都提供了对应的包装类。

  • String() 转换为字符串类型
  • Number() 转换为数字类型
  • Boolean() 转换为布尔类型

每当读取一个基本类型值的时候，后台就会创建一个对应的基本包装类型的对象，从而能调用一些方法来操作这些数据。基本包装类型包括Boolean、Number和String

String类型包含了三个属性和大量的可用内置方法

  • length: 返回字符串的字符长度
  • constructor: 返回创建String对象的函数
  • prototype: 通过添加属性和方法扩展字符串定义

parseInt

将字符串转换为整数。从字符串的开头开始解析，在第一个非整数的位置停止解析，并返回前面读到所有的整数。如果字符串不是以整数开头的，将返回NaN。如：parseInt("10 a") 返回的值是：10，parseInt("a") 返回的值是：NaN。

parseFloat

将字符串转换为浮点数。 从字符串的开头开始解析，在第一个非整数的位置停止解析，并返回前面读到所有的整数。如果字符串不是以整数开头的，将返回NaN。如：parseFloat("10.5 hi") 返回的值是：10.5，parseFloat("a 10.5") 返回的值是：NaN。

eval(不推荐使用，一次编译，一次执行)

将字符串作为javascript表达式进行计算，并返回执行结果，如果没有结果则返回undefined。
