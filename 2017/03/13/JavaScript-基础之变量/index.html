JavaScript 基础之变量

2017年3月12日

JavaScript变量

js中的变量是松散类型，可以用来保存任何类型的数据。变量在第一次用到时就设置于内存中，便于后来在脚本中引用。使用变量之前先进行声明。可以使用 var let 关键字来进行变量声明。

变量命名

变量名包括全局变量，局部变量，类变量，函数参数等等。

变量的命名可以以任意类型前缀+语义化的单词组成，尽量保持变量命名的可读性。例如用驼峰规则命名的变量：sName nAge

前缀规范（根据项目具体需求及其个人编程习惯而定）：

  • s：表示字符串。例如：sName
  • n：表示数字。例如：sAge
  • b：表示逻辑（布尔类型）。例如：bChecked
  • a：表示数组。例如：aList
  • r：表示正则表达式。例如：rEmail
  • f：表示函数。例如：fAdd
  • o：表示其它（以上未涉及到的）。例如：oDate
  • g：表示全局变量。例如：gToken

JavaScript 是一种区分大小写的语言。创建变量（标识符）应该遵循以下几点：

a. 第一个字符必须是字母、数字、下划线或者美元符号开头。

b. 变量在声明的时候不能同JavaScript保留子相同（比如var、function）。

c. 如果变量声明了，但是没有对其复制，该变量的值默认为`undefined`。

变量之间的强制类型转换

在JavaScript中，JavaScript解释器会自动将数据类型之一强制转换为另一种数据类型，然后执行元素。例如：

  • 数字与字符串相加（将数字转换为字符串）
  • 布尔值与字符串相加（将布尔值转换为字符串）
  • 数字与布尔值相加（将布尔值转换为数字）
  • parseInt（将字符串转换为整数）
  • parseFloat（将字符串转换浮点数）
  • 字符串减去一个值会将字符串转换为数值（’123’ - 0）

    变量的生命周期

    当在函数内声明一个变量后，只能在该函数中访问该变量。当退出该函数后，JavaScript垃圾回收机制会销毁该变量。这种变量被成为本地变量（局部变量）。
    如果在函数之外声明了一个变量，则此同级作用域都可访问该变量。
